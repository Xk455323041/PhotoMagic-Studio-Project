## 🎨 详细设计规范

### 1. 页面标题区域
```
专业证件照，标准尺寸和背景
```

**设计规范**:
- **主标题**: `font-size: 2.5rem`, `font-weight: 700`, `color: gray-900`
- **副标题**: `font-size: 1.25rem`, `color: gray-600`, `margin-top: 8px`
- **图标**: 相机图标 📷，`font-size: 2rem`
- **对齐**: 居中对齐
- **间距**: `margin-bottom: 32px`

### 2. 上传区域设计

#### 证件照上传要求
```
支持单张人像照片，系统自动检测人脸
```

**设计规范**:
- **尺寸限制**: 最小 500×500 像素
- **格式支持**: JPG, PNG
- **人脸检测**: 自动验证是否为人像
- **质量检查**: 自动检查清晰度、光线
- **上传反馈**:
  - **成功**: 显示人脸检测框，确认通过
  - **失败**: 提示具体原因（无人脸、质量差等）
  - **建议**: 提供拍摄建议（光线、表情、服装）

#### 上传界面设计
```

│          拍摄建议: 正面、光线充足、表情自然                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**设计规范**:
- **图标**: 相机图标，64px × 64px
- **提示文字**: 清晰说明上传要求
- **人脸检测框**: 上传后自动显示人脸区域
- **质量评分**: 显示图片质量评分（可选）
- **重新拍摄**: 提供重新拍摄建议链接

## 🎯 交互流程设计

### 1. 标准证件照制作流程
```
1. 用户进入页面
2. 上传人像照片（系统自动检测人脸）
3. 选择证件照类型（如：身份证）
4. 选择背景颜色（如：白色）
5. 调整人像位置（居中或微调）
6. 选择美颜选项（可选）
7. 预览证件照效果
8. 选择排版方式（单张/4x6等）
9. 点击"生成证件照"
10. 处理完成后点击"下载"
```

### 2. 批量证件照制作流程
```
1. 点击"批量处理"按钮
2. 选择多张人像照片
3. 统一设置参数（类型、背景等）
4. 点击"开始批量处理"
5. 显示批量处理进度
6. 处理完成后打包下载
```

### 3. 服装模板使用流程
```
1. 点击"服装模板"选项
2. 选择服装类型（如：职业正装）
3. 调整服装位置和大小
4. 调节透明度和颜色
5. 预览最终效果
6. 确认后生成证件照
```

## 📱 响应式设计

### 断点适配策略

#### 移动端 (< 768px)
- **单列布局**: 所有区域垂直排列
- **折叠面板**: 次要设置默认折叠
- **简化控件**: 使用滑块代替复杂选择
- **触摸优化**: 增大按钮和输入框

#### 平板端 (768px - 1023px)
- **双列布局**: 参数设置和预览区域并排
- **适中控件**: 平衡功能和空间
- **保持预览**: 始终显示证件照预览

#### 桌面端 (≥ 1024px)
- **多列布局**: 充分利用屏幕空间
- **丰富控件**: 显示所有高级选项
- **实时预览**: 大尺寸实时预览效果

### 组件响应式变化

### 页面状态
```typescript
interface IDPhotoState {
  // 上传状态
  upload: {
    status: 'idle' | 'uploading' | 'uploaded' | 'error';
    file: File | null;
    previewUrl: string;
    faceDetected: boolean;
    faceBox: { x: number; y: number; width: number; height: number } | null;
    qualityScore: number;
    error: string | null;
  };
  
  // 证件照参数
  parameters: {
    type: 'id_card' | 'passport' | 'driver_license' | 'visa' | 'custom';
    size: {
      width: number; // mm
      height: number; // mm
      dpi: number;
    };
    background: {
      type: 'solid' | 'gradient' | 'custom';
      color: string;
      gradient?: { start: string; end: string; angle: number };
      customImage?: string;
    };
    portrait: {
      position: 'center' | { x: number; y: number };
      zoom: number;
      rotation: number;
      beauty: {
        skinSmooth: number;
        eyeBrighten: number;
        teethWhiten: number;
        removeRedEye: boolean;
      };
    };
    clothing: {
      enabled: boolean;
      type: 'business_formal' | 'business_casual' | 'student_uniform' | 'none';
      opacity: number;
      color: string;
      position: { x: number; y: number };
      scale: number;
    };
    output: {
      format: 'jpg' | 'png';
      quality: number;
      includeOriginal: boolean;
      includeLayout: boolean;
      layoutType: 'single' | '4x6' | '8x10' | 'custom';
    };
  };
  
  // 处理状态
  processing: {
    status: 'idle' | 'processing' | 'completed' | 'error';
    progress: number;
    estimatedTime: number;
    resultUrl: string | null;
    layoutUrl: string | null;
    error: string | null;
  };
}
```
